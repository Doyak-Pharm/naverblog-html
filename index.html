WHO-UMC 판단 기준 요약

✅ Certain

  • ✔ 합리적인 시간적 선후관계
  • ✔ dechallenge 양성 (중단 시 호전)
  • ✔ rechallenge 양성 (재투여 시 재발)
  • ✔ 다른 약물·질환으로 설명 불가
실제 보고율 ~1–5% · rechallenge 요건이 장벽

🔵 Probable

  • ✔ 합리적인 시간적 선후관계
  • ✔ dechallenge 양성
  • ✔ 다른 원인으로 설명하기 어려움
  • — rechallenge 정보 없어도 됨
입원 환자 능동감시에서 ~30–90%

🟣 Possible

  • ✔ 합리적인 시간적 선후관계
  • ⚠ 다른 원인(질환·병용약)으로도 설명 가능
  • ⚠ dechallenge 정보 없거나 불명확
자발보고 시스템에서 가장 많음 ~40–73%

🟡 Unlikely

  • ⚠ 시간적 연관성 희박하거나 불가능
  • ⚠ 다른 원인이 더 설득력 있음
ADR DB에서 가장 적음 ~1–5%

🔴 Conditional / Unclassified

  • ⚠ 이상반응은 있으나 데이터 불충분
  • ⚠ 추가 데이터 수집 후 재평가 필요
판단 보류 상태

⚫ Unassessable / Unclassifiable

  • ✖ 보고 정보가 불충분하거나 모순됨
  • ✖ 추가 데이터로도 평가 불가한 구조적 문제
보고 품질 문제 반영
🔑 PV 실무 핵심 포인트 규제 목적상:
drug-related = Certain + Probable + Possible
not related = Unlikely + Conditional + Unassessable

→ ICSR 보고 의무는 Possible 이상이면 모두 해당됩니다.
